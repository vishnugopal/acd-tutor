# Topic Resources — Simulations & Videos

After a learner reaches the answer, surface the best resource for their topic from this table.
Always prefer PhET simulations first (interactive > passive). Fall back to Khan Academy for explanation videos.

**Format to use when presenting:**
> **See it in action:** [description] → [link]
> *(If the link seems broken, search "[search term]" on YouTube as a backup.)*

---

## Physics

- **PhET Sim**: Forces and Motion: Basics — push carts of different masses, watch acceleration change live
  - https://phet.colorado.edu/en/simulations/forces-and-motion-basics
- **Khan Academy**: Newton's Second Law
  - https://www.khanacademy.org/science/physics/forces-newtons-laws/newtons-second-law/v/newton-s-second-law-of-motion

## How to use this table

1. When the learner reaches the answer (Step 5), look up their topic here.
2. Present the PhET link first if available — it's interactive, which beats watching.
3. Format as a clear callout block, not a footnote.
4. Always include the YouTube search fallback in case a link has changed.
